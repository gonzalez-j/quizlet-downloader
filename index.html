Description

If you've ever found a list of words on Quizlet that you would like to add to notes, you know how just tedious copying and pasting each word and definition can be. Quizlet-downloader automates everything for you. All you do is provide a Quizlet flashcards list URL (Example) and select the separator you would like to use. Building

In your shell of choice, navigate to the directory containing pom.xml. Then, using Maven, run the mvn clean assembly:assembly command. Maven should automatically download dependencies and produce a jar file in the target directory.
